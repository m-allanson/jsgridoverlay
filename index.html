jsgridoverlay by m-allanson

A javascript script for quickly overlaying an image over a web page.

Intended to be an update to http://code.google.com/p/jsgridoverlay/ Once installed, press Ctrl + ; to activate. Drag an image from your machine to the page to load it in. Then toggle with Ctrl + ; or adjust opacity with Ctrl + [ and Ctrl + ].

Install

Include the script at the bottom of your page.
