jsgridoverlay

A javascript script for quickly overlaying an image over a web page.

Intended to be an update to http://code.google.com/p/jsgridoverlay/, written by jeddy3, thegingerbloke and m-allanson

Once installed, press Ctrl + ; to activate. Drag an image from your machine to the page to load it in. Then toggle with Ctrl + ; or adjust opacity with Ctrl + [ and Ctrl + ].

Try it on this page!

Install

Include the following script at the bottom of your page:

https://raw.github.com/m-allanson/jsgridoverlay/master/grid.js

Download

You can download this project here.

You can also clone the project with Git by running: 

$ git clone git://github.com/m-allanson/jsgridoverlay
