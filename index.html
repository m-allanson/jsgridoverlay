A really basic demo page

Usage is as follows:

  • Press Ctrl + ; to enable the overlay
  • Drag an image anywhere onto the green box
  • Increase opacity with Ctrl + ]
  • Decrease opacity with Ctrl + [
  • Toggle the overlay with Ctrl + ;
